# Area & Perimeter ITS (Python)

This repository contains a Python implementation of an Intelligent Tutoring System (ITS) focused on area and perimeter calculations.

## How to run
1.⁠ ⁠Download or clone this repository.
2.⁠ ⁠Open the folder in PyCharm (or any Python IDE).
3.⁠ ⁠Run the main Python file (e.g., ⁠ main.py ⁠).

## Notes
•⁠  ⁠The ITS tracks student progress and provides guidance such as feedback/hints depending on the learner’s performance.
